================================================================================
NVIDIA GEFORCE GAME READY DRIVER RELEASE NOTES (v555.99 / v556.12)
================================================================================
Product Support: GeForce RTX 40 Series, RTX 30 Series, RTX 20 Series
Operating System: Windows 10 64-bit, Windows 11 64-bit
Release Date: 2026

--------------------------------------------------------------------------------
1. GAME READY SUPERIORITY & NEW FEATURES
--------------------------------------------------------------------------------
This new Game Ready Driver provides the optimal gaming experience for the latest 
titles utilizing NVIDIA DLSS 3, DLSS 2, and Reflex technologies. 

- Optimized performance, stability, and frame pacing improvements for:
  * Elden Ring: Shadow of the Erdtree (Adds DLSS 3 Frame Generation support)
  * Senua's Saga: Hellblade II
  * Ghost of Tsushima Director's Cut

--------------------------------------------------------------------------------
2. FIXED ISSUES (Problemas Corrigidos neste Driver)
--------------------------------------------------------------------------------
The following issues have been successfully resolved in this driver release:

- [Elden Ring] Fixed an issue where players experienced random micro-stuttering 
  and sudden frame drops (0.1% lows) during heavy particle effect sequences or 
  boss fights when Ray Tracing was enabled.
- [GeForce Laptops] Fixed a bug causing the GPU to remain stuck in a low-power 
  state (low TGP) after resuming the laptop from Windows Sleep Mode, resulting 
  in severely degraded gaming performance until a system reboot.
- [Ghost of Tsushima Director's Cut] Fixed random desktop crashes (CTD) when 
  combining DLSS Frame Generation with Reflex Boost on specific hardware setups.
- [Notebook Displays] Fixed screen flickering issues on certain high-refresh-rate 
  panels (144Hz/165Hz) used in Lenovo LOQ and Acer Nitro laptops when G-Sync 
  was active in windowed mode.
- [Discord] Video streaming or screen-sharing gameplay might exhibit slight color 
  corruption or artifacting when hardware acceleration is toggled on.

--------------------------------------------------------------------------------
3. OPEN/KNOWN ISSUES (Problemas Conhecidos - Aguardando Correção)
--------------------------------------------------------------------------------
NVIDIA is actively working on fixes for the following user-reported issues:

- [Cyberpunk 2077] Enabling Path Tracing alongside DLSS Ray Reconstruction 
  may cause ghosting artifacts behind fast-moving vehicles. 
  * Temporary Workaround: Disable Ray Reconstruction or reduce motion blur.
- [Asus TUF / Acer Nitro Laptops] Display may show a brief black screen when 
  Advanced Optimus switches dynamically from the Integrated GPU (iGPU) to the 
  Dedicated NVIDIA GPU (dGPU) during game launches.
- [Hardware Monitor Tools] Popular background applications (such as MSI Afterburner) 
  might report incorrect memory clock speeds on specific factory-overclocked 
  GeForce RTX 4060 Laptop GPUs.
